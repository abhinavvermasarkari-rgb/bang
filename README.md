# Prop Hedge Planner

Prop Hedge Planner is a mobile-first calculator for planning prop-firm challenges using hedging and structured risk management. The app turns prop firm rules and a hedging risk plan into daily limits, compliance targets, and phase pass scenarios.

## Features
- Landing page + calculator layout.
- Derived drawdown and target calculations.
- 4-day compliance plan, phase pass estimates, broker deposit range.
- Scenario tree with example outcomes.
- Monte Carlo simulator for rough win-rate estimates.
- Copy plan summary, shareable link, and PDF download.

## Tech Stack
- React + TypeScript (Vite)
- Tailwind CSS
- Recharts
